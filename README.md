# Angular Frontend

This project contains the frontend implementation of a simple Angular application for drawing rectangle SVG figures.

## Getting Started

To get started with the frontend, follow these steps:

### Prerequisites

- [Node.js](https://nodejs.org/) must be installed on your machine.
- Angular CLI must be installed globally. You can install it via npm:
npm install -g @angular/cli


### Installation

1. Clone this repository.
2. Navigate to the `angular-frontend` directory.
3. Run the command `npm install` to install dependencies.
4. Run the command `ng serve` to start the development server.
5. The application will be accessible at `http://localhost:4200`.

## Features

- Draw rectangle SVG figures.
- Resize figures using mouse interactions.
- Display perimeter of the figure near the rectangle.
- Update JSON file with new dimensions after resizing.

## Usage

1. Open your web browser and navigate to `http://localhost:4200`.
2. Draw rectangles and resize them as needed.
3. The perimeter of the figure will be displayed near the rectangle.
4. JSON file will be updated automatically with new dimensions after resizing.
